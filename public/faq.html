How do I stake Wownero?

This staking method is non-custodial, meaning you have compete control over your private keys. 500 WOW is the minimum amount needed to earn rewards from staking.

You will need to have two wallets: a new wallet, which will be used to lock the funds in ("wallet A") and a second wallet with some WOW ("wallet B").
Do not use wallet A for ordinary transactions, think of wallet A more as a "CD account".

  • Download latest Wownero binaries
  • Open terminal, run wownerod to sync with network (takes about 30 minutes)
  • Once synced, open new terminal tab, run wownero-wallet-cli, create a new wallet ("wallet A"), write down the seed and keep it safe offline, just in case your need to recover the wallet!
  • In another tab, open a second wallet with some WOW ("wallet B")
  • From wallet B, transfer the amount you would like to lock + some change to cover the transaction fees (something like 0.05 WOW) to wallet A.
  • Go back to wallet A, wait for the transaction to be confirmed (takes about 15 minutes, but could take up to an hour depending on network activity)
  • Once transaction balance is "unlocked" (type refresh and balance to check status), lock the whole balance of wallet A for 1 year (10,5192 blocks) by sending the balance back to the same wallet. Type address to see walletâ€™s main address and then enter: locked_sweep_all ADDRESS 105192
  • Confirm the transaction (carefully check the address and number of blocks) and then take note of the transaction ID ("TXID"). The WOW is sent back to the wallet A, but the amount will be locked for 1 year. After the time has elapse, the WOW is automatically released and spendable again, at which point you could sweep out the balance or re-lock balance for another year.
  • Now you will need to get the transaction key ("TXKEY") using the TXID: get_tx_key TXID
You are done with the wallet part. Type save and exit.
  • Set up MetaMask on Avalanche
  • Submit staking with the following informations:
    TXID
    TXKEY
    Main address of wallet A
    Avalanche C-chain address
Once the rewards program is launched on 1 January 2022, 1:1 tokens will be sent to your Avalanche address.

Can anybody steal my Wownero?

To staking Wownero you don't have to send private keys; it's only required the transaction key so that the quantity and veracity of the tx can be verified.

Once i locked my wallet, can I unlock it?

No. Please be careful, as once the wallet is locked you will have to wait one year before accessing your funds again.
